# Larissa Belo Studio - Landing Page Premium

Uma landing page elegante, moderna, responsiva (Mobile-First) e de alta conversão projetada para o **Larissa Belo Studio**, focada em serviços de saúde e estética das unhas e pés.

---

## 📸 Demonstração e Visual

O projeto foi construído utilizando um design voltado para o nicho de luxo e bem-estar (Soft Luxury / Wellness):
- **Paleta de Cores**: Fundo em tons de rosa aquarelados e suaves, detalhes e badges em Dourado/Rose Gold, com textos contrastantes em Marsala/Vinho e Cinza Escuro.
- **Tipografia**: Combinação de *Playfair Display* (títulos de seção marcantes), *Dancing Script* (caligrafia fina para sublinhar delicadeza) e *Montserrat* (texto do corpo limpo e legível).
- **Acessibilidade**: Estrutura HTML5 semântica e botões otimizados para toque.

---

## ✨ Funcionalidades

- **Design Responsivo (Mobile-First)**: Otimização total para telas de smartphones, tablets e desktops.
- **Menu Mobile Integrado**: Navegação colapsável estilo hambúrguer com fechamento automático após clique.
- **Efeitos e Micro-Animações**: Transições dinâmicas ao passar o mouse sobre botões e cards de serviços (efeito zoom, sombras graduais).
- **Animações de Entrada (Scroll Reveal)**: Elementos surgem de forma suave conforme o usuário desliza pela página, usando a API nativa `IntersectionObserver` do navegador (alta performance).
- **FAQ em Acordeão**: Seção de perguntas frequentes interativa, poupando espaço na tela móvel.
- **Botão Flutuante do WhatsApp**: CTA persistente com animação pulsante para guiar a usuária ao agendamento.
- **Detox das Unhas (Destaque)**: Card de diferencial exclusivo destacando um serviço padrão incluído em todos os procedimentos.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** (Semântico)
- **CSS3** (Variáveis CSS, Flexbox, Grid e animações keyframes nativas)
- **Vanilla JavaScript** (ES6+, sem dependências externas ou frameworks pesados)
- **Ícones**: Customizados e embutidos via código SVG (alta definição e velocidade de carregamento)

## 🚀 Como Executar Localmente

Como o projeto é construído em código estático puro, não há necessidade de instalação de dependências:

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/larissa-belo-studio.git
   ```
2. Entre no diretório do projeto:
   ```bash
   cd larissa-belo-studio
   ```
3. Abra o arquivo `index.html` em qualquer navegador:
   - Dê dois cliques no arquivo ou abra pelo terminal:
     ```bash
     # No Windows (PowerShell):
     Start-Process index.html
     
     # No macOS / Linux:
     open index.html
     ```

## 🌐 Como Publicar (GitHub Pages)

Este projeto está 100% pronto para ser hospedado gratuitamente no **GitHub Pages**:

1. Crie um repositório no seu GitHub (ex: `larissa-belo-studio`).
2. Suba os arquivos do projeto para o repositório.
3. No GitHub, vá em **Settings** > **Pages**.
4. Em **Build and deployment**, selecione a branch `main` (ou `master`) e a pasta `/ (root)`.
5. Clique em **Save**. O link do seu site estará ativo em alguns minutos!
